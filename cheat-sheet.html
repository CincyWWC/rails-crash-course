Rails

Models
Represent the objects in your system.
  • Name is singular — Post
Controllers
Handles endpoints for the application.
  • Name is plural — PostsController
Method Route Used For
index /posts Lists all models
show /post/:id Displays single model
new /post/new Displays form to create new model
create /posts Creates new model
edit /posts/:id/edit Displays form to update existing model
update /posts/:id Updates existing model
destroy /posts/:id Deletes model
Views
Handles the HTML view for each endpoint in the application.
  • Name matches controller and function — views/posts/index.html.erb
