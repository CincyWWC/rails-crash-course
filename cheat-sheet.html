Rails

Models
Represent the objects in your system.
  • Name is singular — Post
Controllers
Handles endpoints for the application.
  • Name is plural — PostsController
  • Standard CRUD endpoints are
    • index — lists all models
    • show — displays individual model with params[:id]
    • new — displays a form to create a new model
    • create — creates a new model with the passed params[:model_name]
    • delete — destroys individual model
Views
Handles the HTML view for each endpoint in the application.
  • Name matches controller and function — views/posts/index.html.erb
